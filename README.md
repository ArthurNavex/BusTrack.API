# BusTrack.API

API para consulta de linhas, pontos, trajetos e horários de ônibus.

## Tecnologias

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server

## Objetivo

Projeto desenvolvido para estudo de desenvolvimento backend e apresentação de portfólio.

## Funcionalidades planejadas

- Cadastro de usuários
- Consulta de linhas
- Consulta de pontos
- Trajetos
- Linhas favoritas
